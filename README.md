# Responsive Sign-Up Form

A clean and responsive **HTML and CSS sign-up form** project. This project demonstrates front-end form design, responsive layout, input validation, password confirmation, and user-friendly styling.

---

## GitHub Description

A responsive HTML and CSS sign-up form with built-in validation, password confirmation, split-screen layout, and mobile-friendly design.

---

## Project Overview

This project is a front-end sign-up page created using **HTML5** and **CSS3**. It includes a modern split-screen layout with an image and logo section on the left side and a registration form on the right side.

The form allows users to enter their personal details such as first name, last name, email, phone number, password, and confirm password. It uses built-in HTML validation to check required fields, email format, password strength, and password matching.

This project was created to practice form structure, CSS layout, validation, responsive design, and user interface styling.

## Password Validation Rules

The password must contain:

```text
At least 8 characters
At least one uppercase letter
At least one lowercase letter
At least one number
At least one special character
```

The confirm password field checks whether it matches the original password field.

---

## Responsive Design

The project is responsive for different screen sizes.

On larger screens, the layout is displayed in two sections:

```text
Left side: image and logo
Right side: sign-up form
```

On smaller screens, the layout changes into a vertical structure so the form remains readable and user-friendly on mobile devices.
